Introduction

This collection of guided code walk-throughs is an introduction to the functonal programming language Elixir. Elixir is based on another really cool language called Erlang. The Erlang VM is a battle tested virtual machine that excels at concurrency.

I begin by covering the basics of the Elixir programming language. Next, I cover the concurrency in Elixir.

If you'd like to stay connected and get updates when I add new playbacks you can follow me on twitter: @markm208.
